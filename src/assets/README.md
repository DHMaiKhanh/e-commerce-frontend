# Assets

Tài nguyên tĩnh được bundle qua Vite. Phân chia thư mục:

- `images/` — Ảnh nội dung (jpg, png, webp, svg)
- `icons/` — Icon SVG inline hoặc sprite
- `fonts/` — Font files (woff2)

Tệp lớn (>50KB) hoặc tài nguyên public không cần xử lý nên đặt vào thư mục `public/` ở root project thay vì `src/assets/`.
